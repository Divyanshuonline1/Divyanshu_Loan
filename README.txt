# 💰 Loan Enquiry Website

A responsive and professional **Loan Enquiry Website** developed using **HTML, CSS, JavaScript, and PHP**. This platform allows users to explore various loan types, calculate EMIs, submit enquiry forms, and subscribe to newsletters. It also includes an admin dashboard to manage submitted data.

## 🛠️ Technologies Used

- **Frontend:**
  - HTML5
  - CSS3 (Flexbox, Media Queries)
  - JavaScript (Form validation, interactivity)
  
- **Backend:**
  - PHP (Form handling, Admin Panel)
  
- **Database:**
  - MySQL (Data storage)

 🔑 Features

- 🧾 **Multi-step loan enquiry form with validation**
- 💸 **EMI calculator**
- 📧 **Newsletter subscription form**
- 🔐 **Simple Admin Dashboard to view enquiries**
- 📱 **Fully responsive design**
- 📂 **Database integration with PHP + MySQL**
